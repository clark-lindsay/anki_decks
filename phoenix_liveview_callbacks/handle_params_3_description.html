handle_params(unsigned_params, uri, socket)

(optional)

Invoked after mount and whenever there is a live patch event.

It receives the current {{c1::params}}, including {{c1::parameters}} from the router, the current {{c1::uri}} from the client and the {{c1::socket}}. It is invoked {{c2::after mount}} or whenever there is a live navigation event caused by {{c2::push_patch/2}} or {{c2::Phoenix.LiveView.Helpers.live_patch/2}}.

It must always return {:noreply, socket}, where :noreply means no additional information is sent to the client.
