render(assigns :: Phoenix.LiveView.Socket.assigns()) ::
  Phoenix.LiveView.Rendered.t()

If you define this function, it must return a template defined via {{c1::the Phoenix.LiveView.Helpers.sigil_H/2}}.

If you don't define this function, LiveView will attempt to {{c1::render a template in the same directory as your LiveView}}. For example, if you have a LiveView named MyApp.MyCustomView inside lib/my_app/live_views/my_custom_view.ex, Phoenix will {{c1::look for a template at lib/my_app/live_views/my_custom_view.html.heex}}.
