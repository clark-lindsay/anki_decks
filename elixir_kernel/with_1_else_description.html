An else option can be given to modify what is being returned from with in the case of a failed match.

The else block works like a case clause: it can {{c1::have multiple clauses}}, and the first match {{c1::will be used}}. Variables bound inside with are {{c1::not available in the else block}}.

If an else block is used and there are no matching clauses, {{c2::a WithClauseError exception is raised}}.
