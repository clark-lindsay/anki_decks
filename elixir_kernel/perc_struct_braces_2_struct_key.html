%struct{}

(macro) 
defmodule User do
  defstruct name: "john", age: 27
end

Now a struct can be created as follows:

%User{}

Underneath a struct is just a map with a {{c1:::__struct__}} key pointing to the {{c1::User}} module:

%User{} == %{{{c1::__struct__: User}}, name: {{c1::"john"}}, age: {{c1::27}}}
