left && right

(macro)

Boolean "and" operator.

Provides a short-circuit operator that evaluates and returns the second expression only if the first one evaluates to a {{c1::truthy}} value (neither {{c1::false}} nor {{c1::nil}}). Returns the first expression otherwise.

{{c2::Not allowed}} in guard clauses.

Note that, unlike and/2, this operator accepts any expression as the first argument, not only {{c2::booleans}}.
