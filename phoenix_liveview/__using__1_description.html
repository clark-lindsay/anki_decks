__using__(opts)

(macro)

Uses LiveView in the current module to mark it a LiveView.

use Phoenix.LiveView,
  {{c2::namespace}}: MyAppWeb,
  {{c2::container}}: {:tr, class: "colorized"},
  {{c2::layout}}: {MyAppWeb.LayoutView, "live.html"}

Options

  • {{c2:::namespace}} - {{c1::configures the namespace the LiveView is in}}
  • {{c2:::container}} - configures t{{c1::he container the LiveView will be wrapped in}}
  • {{c2:::layout}} - configures {{c1::the layout the LiveView will be rendered in}}
